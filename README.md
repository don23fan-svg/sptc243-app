# AI & Emerging Technologies in Sports Communication

Interactive learning companion for SPTC 243 at Montclair State University.

## 🚀 Deploy to GitHub Pages (Step-by-Step)

### 1. Create a GitHub account (if you don't have one)
- Go to [github.com](https://github.com) and sign up (free)

### 2. Create a new repository
- Click the **+** button in the top-right → **New repository**
- Name it something like `sptc243-app`
- Make it **Public**
- Do NOT check "Add a README" (we already have one)
- Click **Create repository**

### 3. Upload the project files
- On your new empty repo page, click **"uploading an existing file"** link
- Drag and drop ALL the files/folders from this project:
  - `index.html`
  - `package.json`
  - `vite.config.js`
  - `README.md`
  - `src/` folder (with `main.jsx` and `App.jsx` inside)
  - `.github/` folder (with the workflow file inside)
- Click **Commit changes**

### 4. Enable GitHub Pages
- Go to your repo **Settings** → **Pages** (in left sidebar)
- Under **Source**, select **GitHub Actions**
- That's it — the workflow will run automatically

### 5. Wait ~2 minutes, then visit your site
- Go to **Actions** tab to watch the build progress
- Once complete, your site will be live at:
  `https://YOUR-USERNAME.github.io/sptc243-app/`

## 🔧 Local Development (optional)

If you want to run it on your computer:

```bash
npm install
npm run dev
```

Then open http://localhost:5173
